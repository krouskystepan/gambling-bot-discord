src/app/commands
Permission tiers: (misc) = players | (mod) = moderators/admins

├── (misc)  — normal users — global player commands
│   ├── (atm)
│   │   ├── deposit.ts
│   │   │   └─ /deposit – Deposit money to your account.
│   │   │      ├─ amount – The amount you want to deposit (e.g., 1000, 2k, 10.5k).
│   │   │      ├─ account – The account from which you are sending money.
│   │   ├── register.ts
│   │   │   └─ /register – Register yourself in the system.
│   │   └── withdraw.ts
│   │       └─ /withdraw – Withdraw money from your account.
│   │          ├─ amount – The amount you want to withdraw (e.g., 1000, 2k, 10.5k).
│   │          ├─ account – The account you want to send the money to.
│   ├── (casino)
│   │   ├── (other)
│   │   │   ├── blackjack.ts
│   │   │   │   └─ /blackjack – Start a game of blackjack. You can hit, stand, or double down.
│   │   │   │      ├─ bet – Place a bet (e.g., 1000, 2k, 4.5k).
│   │   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │   └── rps.ts
│   │   │       └─ /rps – Play rock, paper, scissors with another user.
│   │   │          ├─ player – The user you want to play against.
│   │   │          ├─ bet – Enter a bet (e.g. 1000, 2k, 4.5k).
│   │   ├── coin-flip.ts
│   │   │   └─ /coin-flip – Flip a coin!
│   │   │      ├─ bet – Place a bet (e.g., 1000, 2k, 4.5k).
│   │   │      ├─ side – Choose the coin side.
│   │   │      ├─ flips – Number of flips.
│   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │      ├─ skip-animations – Skip game animations for faster results.
│   │   ├── dice.ts
│   │   │   └─ /dice – Play a dice game!
│   │   │      ├─ bet – Place a bet (e.g., 1000, 2k, 4.5k).
│   │   │      ├─ side – Choose a dice side.
│   │   │      ├─ rolls – Number of rolls.
│   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │      ├─ skip-animations – Skip game animations for faster results.
│   │   ├── goldenjackpot.ts
│   │   │   └─ /goldenjackpot – Try your luck at the Golden Jackpot HUGEx!
│   │   │      ├─ bet – Place a bet (e.g., 1000, 2k, 4.5k).
│   │   │      ├─ entries – Number of entries (max is ${GOLDEN_JACKPOT_MAX_ENTRIES}).
│   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │      ├─ skip-animations – Skip game animations for faster results.
│   │   ├── lottery.ts
│   │   │   └─ /lottery – Play the lottery! Pick 4 numbers and see if you win.
│   │   │      ├─ bet – Your bet amount (e.g., 100, 1k, 5k).
│   │   │      ├─ numbers – Pick 4 numbers between 1-50, separated by commas (e.g., 3, 14, 25, 38).
│   │   │      ├─ entries – Number of entries.
│   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │      ├─ skip-animations – Skip game animations for faster results.
│   │   ├── plinko.ts
│   │   │   └─ /plinko – Drop balls down the Plinko board!
│   │   │      ├─ bet – Bet per ball (e.g., 1000, 2k, 4.5k).
│   │   │      ├─ balls – How many balls to drop.
│   │   │      ├─ show-balance – Displays the current balance (VISIBLE TO EVERYONE).
│   │   │      ├─ skip-animations – Skip animations for faster results.
│   │   ├── roulette.ts
│   │   │   └─ /roulette – Play Mini Roulette with multiple bets!
│   │   │      ├─ bets – Your bets (e.g., 100 red, 50 17, 200 d2, 75 c1)
│   │   │      ├─ spins – Number of spins.
│   │   │      ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │   │      ├─ skip-animations – Skip game animations for faster results.
│   │   └── slots.ts
│   │       └─ /slots – Spin the slot machine!
│   │          ├─ bet – Place a bet (e.g., 1000, 2k, 4.5k).
│   │          ├─ spins – Number of spins.
│   │          ├─ show-balance – Displays the current balance (WARNING: VISIBLE TO EVERYONE)!
│   │          ├─ skip-animations – Skip game animations for faster results.
│   └── (utils)
│       ├── balance.ts
│       │   └─ /balance – Check your current balance (only you can see this).
│       ├── bonus.ts
│       │   └─ /bonus – Daily bonus system with streaks.
│       │      ├─ claim – Claim your daily bonus
│       │      ├─ check – Check your streak and next bonus
│       ├── ping.ts
│       │   └─ /ping – Check the bot latency.
│       └── vip.ts
│           └─ /vip – VIP management commands.
│              ├─ info – Show VIP info.
│              ├─ buy – Purchase a VIP room for a specified duration.
│              ├─ extend – Extend your current VIP duration.
│              ├─ add-member – Add a user to your VIP room.
│              ├─ remove-member – Remove a user from your VIP room.
└── (mod)  — moderators — admin / dev-guild commands
    ├── (auth)
    │   ├── force-register.ts
    │   │   └─ /force-register – Force register a user.
    │   │      ├─ user – The user you want to register.
    │   └── force-unregister.ts
    │       └─ /force-unregister – Unregister a user (delete from DB).
    │          ├─ user-id – The ID of the user you want to unregister.
    ├── (config)
    │   ├── setup-atm.ts
    │   │   └─ /setup-atm – Manage ATM actions and logs channels.
    │   │      ├─ add-actions – Set a channel for ATM transactions (deposits and withdrawals).
    │   │      ├─ remove-actions – Remove a channel for ATM transactions using its ID (deposits and withdrawals).
    │   │      ├─ add-logs – Set a channel for ATM logs (transaction logs).
    │   │      ├─ remove-logs – Remove a channel for ATM logs using its ID (transaction logs).
    │   ├── setup-casino.ts
    │   │   └─ /setup-casino – Manage the casino channels.
    │   │      ├─ add – Set a channel for using casino bets.
    │   │      ├─ remove – Remove a channel for using casino bets by ID.
    │   │      ├─ add-announcements – Set a channel for public big-win announcements.
    │   │      ├─ remove-announcements – Remove the public big-win announcements channel.
    │   ├── setup-manager.ts
    │   │   └─ /setup-manager – Manage the manager role.
    │   │      ├─ set-role – Set the manager role.
    │   │      ├─ remove – Remove the manager role using its ID.
    │   ├── setup-prediction.ts
    │   │   └─ /setup-prediction – Manage channels for predictions (actions & logs).
    │   │      ├─ add-actions – Add a channel where predictions can be used.
    │   │      ├─ remove-actions – Remove a channel from prediction actions.
    │   │      ├─ add-logs – Set a channel for prediction logs (results).
    │   │      ├─ remove-logs – Remove the prediction logs channel.
    │   ├── setup-raffle.ts
    │   │   └─ /setup-raffle – Manage channels for raffles (actions & logs).
    │   │      ├─ add-actions – Add a channel where raffles can be used.
    │   │      ├─ remove-actions – Remove a channel from raffle actions.
    │   │      ├─ add-logs – Set a channel for raffle logs (results).
    │   │      ├─ remove-logs – Remove the raffle logs channel.
    │   └── setup-vip.ts
    │       └─ /setup-vip – Manage VIP settings.
    │          ├─ add-category – Set a category for VIP rooms.
    │          ├─ remove-category – Remove the VIP category.
    │          ├─ add-owner-role – Set a owner role for VIP users.
    │          ├─ remove-owner-role – Remove the owner VIP role.
    │          ├─ add-member-role – Set a member role for VIP users.
    │          ├─ remove-member-role – Remove the member VIP role.
    │          ├─ set-price-per-day – Set the price per day for VIP access.
    │          ├─ set-create-price – Set the one-time price for creating a VIP room.
    │          ├─ set-add-member-price – Set the one-time price for adding a member to room.
    ├── (dev)
    │   ├── mock-db.ts
    │   │   └─ /mock-db – Seed the database with realistic mock data for Admin testing (dev only).
    │   └── clear-mock-db.ts
    │       └─ /clear-mock-db – Remove mock / operational guild data (keeps guild config). Dev only.
    │          ├─ all – Fill all collections (users, txs, ATM, raffles, predictions, VIP).
    │          ├─ entity – Fill a single collection.
    │          ├─ custom – Fill all collections with custom counts.
    ├── (events)
    │   ├── prediction.ts
    │   │   └─ /prediction – Manage predictions.
    │   │      ├─ create – Create a new prediction.
    │   │      ├─ end – End an active prediction so no more bets can be placed.
    │   │      ├─ payout – Pay out winners of a prediction.
    │   │      ├─ cancel – Cancel a prediction and refund all bets.
    │   │      ├─ check – Check the status of a prediction.
    │   └── raffle.ts
    │       └─ /raffle – Manage global ticket raffles.
    │          ├─ create – Create a new scheduled raffle.
    │          ├─ cancel – Cancel raffle and refund all tickets.
    │          ├─ check – Check how many tickets each user has bought in a raffle.
    └── (utils)
        ├── casino-games.ts
        │   └─ /casino-games – Show available casino games and how to use them.
        ├── casino-info.ts
        │   └─ /casino-info – Get information about the casino.
        │      ├─ games – Show information about casino games
        │      ├─ config – Show server casino configuration
        │      ├─ admin – Show administrator-only information
        ├── manage-balance.ts
        │   └─ /manage-balance – Manage user balances.
        │      ├─ deposit – Add money to a user.
        │      ├─ withdraw – Remove money from a user.
        │      ├─ add-bonus – Give a bonus to a user.
        │      ├─ remove-bonus – Remove bonus funds from a user.
        │      ├─ check – Check a user’s balance.
        │      ├─ reset – Reset a user’s balance to zero and remove all their transactions in this server.
        ├── manage-vip.ts
        │   └─ /manage-vip – Admin commands to manage VIP rooms.
        │      ├─ create-room – Create a VIP room for a user.
        │      ├─ remove-room – Remove a user’s VIP room.
        │      ├─ extend-room – Extend a user’s VIP room.
        │      ├─ add-member – Add a member to someone’s VIP room.
        ├── money-manager.ts
        │   └─ /money-manager – Create an embed for manage balance.
        │      ├─ give-balance – Create an embed for giving money.
        │      ├─ reset-balance – Create an embed for resetting money.
        ├── role-info.ts
        │   └─ /role-info – Display information about a role.
        │      ├─ role – The role you want to get information about.
        └── who-is.ts
            └─ /who-is – Get information about a user.
               ├─ user – The user you want to get information about.
